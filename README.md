## Getting Started

First, install dependencies:

```bash
    npm install
```

Then run the server:

```bash
    npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
Or, if you want a quicker one, go on [CycleMap](https://cycle-map-challenge.vercel.app/networks)
